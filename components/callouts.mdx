# Callouts

Use callouts to add eye-catching context to your content. Callouts help draw attention to important information and provide visual emphasis for different types of content.

## Note Callout

<Note>This adds a note in the content</Note>

```jsx
<Note>This adds a note in the content</Note>
```

## Warning Callout

<Warning>This raises a warning to watch out for</Warning>

```jsx
<Warning>This raises a warning to watch out for</Warning>
```

## Info Callout

<Info>This draws attention to important information</Info>

```jsx
<Info>This draws attention to important information</Info>
```

## Tip Callout

<Tip>This suggests a helpful tip</Tip>

```jsx
<Tip>This suggests a helpful tip</Tip>
```

## Check Callout

<Check>This brings us a checked status</Check>

```jsx
<Check>This brings us a checked status</Check>
```

## Danger Callout

<Danger>This is a danger callout</Danger>

```jsx
<Danger>This is a danger callout</Danger>
```

## Usage Guidelines

Callouts are perfect for:
- **Notes**: General information or explanations
- **Warnings**: Important cautions or potential issues
- **Info**: Key details that need attention
- **Tips**: Helpful suggestions or best practices
- **Check**: Confirmation or success states
- **Danger**: Critical warnings or error states

Choose the appropriate callout type based on the importance and nature of your message to ensure your documentation is both informative and visually appealing.
